# TDG40k

Developer: **Auda29**

> `TDG40k` is currently a **working title**.

TDG40k is an early-stage **tower defense game project** inspired by the readability, progression, and upgrade depth of games like Bloons TD6, with an **original grim sci-fi warfare theme**.

## Project status
This repository is currently in the **concept / pre-production** phase.

Current focus:
- define the core gameplay loop
- establish project vision and scope
- document tower, enemy, and progression concepts
- prepare the repository for future implementation

## Vision
The game aims to combine:
- satisfying tower defense gameplay
- strong upgrade paths and build variety
- clear combat readability
- dark, militaristic sci-fi atmosphere
- an original setting inspired by, but not copying, existing genre works

## Current design direction
Planned early prototype / MVP direction:
- singleplayer first
- top-down or isometric presentation
- fortress / outpost defense fantasy
- high combat readability
- strong tower roles and upgrade decisions

For the current MVP definition, see:
- `docs/mvp-scope.md`
